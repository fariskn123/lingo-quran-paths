# 📿 Welcome to Qur’ānLingo – Learn Qur’anic Arabic the Duolingo Way

Qur’ānLingo is a mobile-first, spiritually immersive web app that teaches Qur’anic Arabic through high-impact vocabulary, flashcards, sentence translation, and gamified XP progression — inspired by the Duolingo learning experience.

---

## ✨ Features

- 🟢 **Level-based learning path** (Power Words → Judgment Day → Prayer → Abstract)
- 📚 **Manually curated lessons** — each with real Qur’anic vocabulary, meanings, and examples
- 🎴 **Flashcards** + ✍️ **Sentence Builders**
- 🔥 XP tracking, streaks, and lesson victory screens
- 🎨 Mobile-first design using Tailwind CSS + Framer Motion

---

## 📁 Project Structure

This version uses will either use manual lesson files, database, or dynamic loader.

This project is built with:
